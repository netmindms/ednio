Detailed Description

Event Driven Task frame class.

Author
netmind
Remarks
EdTask is basically a thread with a event loop.
It works asynchronously by using non-block io. All codes in EdTask should not be blocked with "best effort". Any code to be blocked will cause delay other events on the same task(thread).
