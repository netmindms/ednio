About

Ednio is a event driven, non-blocked, asynchronous task library, which is free, open source.
This library can be used for system applicaton, network commnunication service.

General Informations

Synchronous Model and Asynchronous Model

In general, there are two model in network programming model, which are synchronous and asynchronus models. In synchronous model, the thread as many as network connections are needed. In asynchronous mode, single thread can process many connections.


Features

  • Native C++ library for Linux
  • Provide task having event loop.
  • Asynchronous network API.
  • Asynchronous timer and IPC API.
  • Asynchronous Curl wrapper API
  • Asyncrhonous OpenSSL wrapper API
  • Asyncrhonous MariaDB wrapper API
